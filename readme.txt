Adds element numbering to Wordpress with tinyMCE integration.

== Description ==

Add paragraph and other element numbering to Wordpress content.
Automatic, customisable numbering is supported through TinyMCE integration.
Insert numbering as shortcodes or plain html.

[dudeyjon.com/paranum](http://dudeyjon.com/paranum/)

== Installation ==

1. Upload the 'paranum' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress.

Numbering can now be added to posts and pages using the new button provided on the TinyMCE editor.
